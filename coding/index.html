Instagram Caption Generator

This is a project to try and create a Instagram caption generator by training a neural network to associate phrases/captions with certain objects within the image provided. This process will involve web scraping (to get training data), some neural model to convert the images to a word description (image classification) and then a neural model to convert an image to an actual caption with an intermediary step of realizing what the image consists of.

Bilingualism Transfer Learning

This project explores the idea of bilingual speakers being better at language-related tasks (specifically novel word processing). It then tries to expand this to language-related tasks for neural models, which is done through transfer learning. A model will be trained on a novel word processing task in a more morphologically rich language (in this case, Finnish) before it is then trained and tested on another language (morphologically poorer, in this case, English).

Spotify Music Recommender

This project is inspired by research that states that on average, an individual's music taste is developed between the ages of 11 and 16. Less formally, a sense of nostalgia brought by a song can sometimes influence someone to enjoy it. This means that by using someone's age, you should be able to set a baseline music taste from which you can use Spotify's music recommendation techniques on to find similar songs. These techniques, which I'll try to recreate include using convolutional neural architectures to learn audio features and text mining for sentiment terms to use for bucketing and describing certain songs.

Musicify

This project is a progressive web app that revolves around the idea of sharing the meaning behind music with friends. I like to think of it as Genius but more personalized for someone and his/her friends. It allows for you to not only share the songs but also annotate what the lyrics mean (editable from both ends).

Real World Transfer of Learning

Transfer learning is the concept that you can utilize skills from one task on an orthogonal task. For example, knowing how to rollerblade might help you with knowing how to ice skate even though they’re completely different sports/activities. I look at the wiki pages of athletes and try to draw connections between the sports they played when they were younger and their primary sport to see if there’s a link between them.

Subgenre Clustering for Movie Recommendation Systems (PennApps 2019)

By using the scripts of these movies, I make new clusters of subgenres using Gaussian Mixture models which will help yield improvements in collaborative filtering for movie recommendation systems.
