Current Projects (WIP)

Instagram Caption Generator

This is a project to try and create a Instagram caption generator by training a neural network to associate phrases/captions with certain objects within the image provided. This process will involve web scraping (to get training data), some neural model to convert the images to a word description (image classification) and then a neural model to convert an image to an actual caption with an intermediary step of realizing what the image consists of.

Bilingualism Transfer Learning

This projects explores the idea of bilingual speakers being better at language-related tasks (specifically novel word processing). It then tries to expand this to language-related tasks for neural models, which is done through transfer learning. A model will be trained on a novel word processing task in a more morphologically rich language (in this case, Finnish) before it is then trained and tested on another language (morphologically poorer, in this case, English).

View

Tetris

This is a simple implementation of the game, Tetris, which is to be used for training a model on how to best play Tetris. The game itself is coded using PyGame, a Python wrapper for the SDL library, designed for writing video games. The model will be trained on a variety of heuristics to allow the model to learn different playing techniques and strategies.
